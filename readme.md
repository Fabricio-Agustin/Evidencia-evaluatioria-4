## 📖 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una clase `Helicoptero` utilizando Programación Orientada a Objetos (POO) en Python.

El objetivo es representar el comportamiento de un helicóptero mediante atributos y métodos que permitan gestionar 
operaciones básicas como el despegue, vuelo y aterrizaje, aplicando conceptos fundamentales de encapsulamiento, abstracción 
y lógica de programación.

Además, el proyecto incorpora pruebas unitarias para validar el funcionamiento de la clase y un diseño de base de datos relacional para almacenar información relacionada con los helicópteros.
